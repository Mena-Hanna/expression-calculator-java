# Test Cases for Expression Calculator

# Basic operations
2 + 3 * 4            => 14
(2 + 3) * 4          => 20
2 + 3 + 4            => 9
5 - 3 + 2            => 4

# Parentheses handling
(8 + 2 * 5) - (9 / 3 + 4) + (6 * (7 - 2)) / 5    => 17
(10 / 2) + 6        => 11
(3 + (4 * 5))       => 23

# Negative numbers
-5 + 2              => -3
-3 * (4 - 6)        => 6
-2 + 3              => 1

# Decimals
3.5 * 2             => 7
5.0 + 2.5           => 7.5
(10.5 / 2) + 3.0    => 8.25

# Exponentiation 
2 ^ 3               => 8
3 ^ 2               => 9
(4 / 1.5) ^ 0.5 * 3 ^ 1.2    => 6.10281
5 ^ 3               => 125

# Edge cases
( 10 + 3 ) * 2      => 26
( 2 + 3 ) / ( 4 - 1 ) => 1.666667
3 + 0               => 3
0 * 5               => 0
( 1 + 2 ) + ( 3 + 4 )  => 10
